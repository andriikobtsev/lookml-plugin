LookML Support

Looker Modeling Language for IntelliJ IDEA, PyCharm and other JetBrains IDEs. Highlighting, completion, validation and formatting - right in your editor.

Get it on the JetBrains Marketplace Source on GitHub

What it does

  • Syntax highlighting for .lkml / .lookml - keywords, strings, SQL blocks, field references.
  • Code navigation - go-to-definition, find usages, rename, and go-to-implementation/super for fields and views, across extended and refined views and YAML dashboards.
  • Code completion for traditional LookML and YAML dashboards (150+ dashboard properties), plus in-sql: field suggestions.
  • Validation - syntax errors plus schema-backed checks for YAML dashboard properties.
  • Formatting - Reformat Code for LookML and a canonical YAML dashboard rewriter.
  • Editing aids - folding, commenting, brace matching, manifest file support.
